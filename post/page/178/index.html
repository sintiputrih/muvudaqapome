Unified Memory for CPU/GPU in 2013, HSA GPUs in 2014

published on July 16, 2024
AMD's CTO Mark Papermaster just put up this slide that shows its HSA (Heterogeneous Systems Architecture) roadmap through 2014. This year we got Graphics Core Next, but next year we'll see a unified address space that both AMD CPUs and GPUs can access (today CPUs and GPUs mostly store separate copies of data in separate memory spaces). In 2014 AMD plans to deliver HSA compatible GPUs that allow for true heterogeneous computing where workloads will run, seamlessly, on both CPUs and GPUs in parallel. Read More...
